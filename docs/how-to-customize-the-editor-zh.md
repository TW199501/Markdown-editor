##### 入门指南 / 编辑器自定义

## 如何自定义编辑器
您可以使用 CSS 变量来使编辑器内容符合您的需求

| **变量** | **描述** | **CSS 属性类型** | **默认值** |
| :---:  | :---: | :---: | :---: |
| `--g-md-toolbar-padding` | 工具栏内边距 | padding | 0px |
| `--g-md-toolbar-sticky-padding` | 粘性模式下工具栏内边距 | padding | -4px |
| `--g-md-toolbar-sticky-inset` | 粘性模式下工具栏插入 | inset | -4px |
| `--g-md-toolbar-sticky-offset` | 粘性模式下工具栏偏移 | top | 0px |
| `--g-md-toolbar-sticky-border` | 粘性模式下工具栏边框 | border | 1px solid var(--g-color-line-generic-solid) |
| `--g-md-editor-padding` | 编辑器内容内边距 | padding | 0px |
